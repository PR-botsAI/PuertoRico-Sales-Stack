# 07 — Quick Reference Matrix

*One-page cheat sheet for the buyer-profile section. Use this in the field; the depth is in the other six files.*

## The 30-second pre-call profile

Before any consequential call, answer these six questions in your head:

1. **Wealth tier?** — Affluent / Middle / Working / Subsistence
2. **Need level?** — Safety / Belonging / Esteem / Self-actualization
3. **Top need (Robbins)?** — Certainty / Variety / Significance / Connection / Growth / Contribution
4. **DISC type?** — D / I / S / C (Driver / Influencer / Steady / Conscientious)
5. **Generation?** — Boomer / Gen X / Millennial / Gen Z
6. **PR archetype?** — Dueño / Profesional / Don-Doña / Diaspora / Act 60 / Abuela / Resolvedor / Cangrejo / Hustler / Trabajador / Buscavidas

Thirty seconds. Now you have a working hypothesis.

## Six Human Needs — buyer signal words

| Need | They say… | Sell… |
|---|---|---|
| Certainty | Reliable, proven, safe, guaranteed | Track record, warranties, slow rollout |
| Variety | New, different, latest, unique | First-access, limited editions, novelty |
| Significance | Best, premier, executive, VIP | Exclusivity, named-tier, status |
| Connection | Family, we, together, share | Family experiences, community, shared products |
| Growth | Learn, develop, master, improve | Education, coaching, mastery programs |
| Contribution | Impact, legacy, give back, matter | Legacy products, philanthropy, mentorship |

## Red flag signals — stop and reassess

If you observe any of these, **pause your default pitch:**

- Buyer references family decision-makers you haven't met → map the full committee
- Buyer keeps switching languages mid-conversation → read the cultural signal; mirror correctly
- Buyer mentions a specific competitor by name early → there's a Blocker or incumbent investment
- Buyer is in *cangrejo* mode → acknowledge concerns; don't push
- Buyer is dressed/situated above their stated income level → wealth tier may be different than expected (could be down-class signaling, could be up-class genuine)
- Buyer asks no questions at all → they're not engaged; reset
- Buyer asks all questions at once → likely a Conscientious type collecting full data

## When in doubt — the universal moves

If you can't identify the profile, default to these moves — they work across nearly all PR buyer types:

1. **Open with relational warmth** (90 seconds; ask about family / day / area)
2. **Listen more than talk** (first 5 minutes especially)
3. **Match their language register** (Spanish/English, formal/casual)
4. **Match their pace** (fast with Ds, slow with Ss)
5. **Frame value in their words** (use what they say their problem is)
6. **Honor their decision process** (don't pressure if they don't want to decide today)
7. **Be honest about tradeoffs** (over-promising kills more PR deals than under-promising)
8. **Follow through fast on commitments** (slow follow-through is the #1 PR deal-killer)
