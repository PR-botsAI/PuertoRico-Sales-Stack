# Contributing

This archive grows. Pull requests welcome from anyone working in or selling into the Puerto Rico market.

## What we accept

- New book summaries (must follow the schema in `BOOKS/README.md`)
- New speaker profiles (must follow the schema in `SPEAKERS/README.md`)
- New strategy frameworks (synthesis of multiple sources, not single-book summaries)
- PR-specific applications, scripts, or case studies for `PR-APPLICATION/`
- Corrections, clarifications, broken-link fixes

## What we don't accept

- Affiliate-link spam
- Single-author opinion essays without source grounding
- Hype content ("this book changed my life") — describe the frameworks, not the feeling
- AI-generated text dumps without human curation and source verification

## Schema discipline

The value of this archive is its uniformity. Every book file uses the same eight section headers. Every speaker file uses the same seven. If you contribute a file that breaks the schema, expect a polite request to reformat.

## Source standards

- Quotations from copyrighted works: under 25 words, attributed, used for commentary.
- Claims about a speaker's biography or methodology: linked to a primary source where practical.
- Statistics and case studies: linked to the original publication, not to a blog rehash.

## How to submit

1. Fork the repo.
2. Branch named `add/book-title-slug` or `add/speaker-slug` or `add/strategy-slug`.
3. One file per PR where possible.
4. PR description: which gap in the archive does this fill?

Maintainer review will check schema, sourcing, and PR-applicability. Expect a response within a week.
