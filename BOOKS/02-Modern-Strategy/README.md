## How to use this section

If you have **30 days**, read in this order:
1. *$100M Offers* — fix what you're selling.
2. *Fanatical Prospecting* — fix your pipeline.
3. *Never Split the Difference* — fix your conversations.
4. *$100M Leads* — fix your top-of-funnel.
5. *The Challenger Sale* — fix how you create value mid-funnel.

If you sell **enterprise/complex deals**: skip straight to *Strategic Selling*, *Mastering the Complex Sale*, *Selling to VITO*, then *The Challenger Sale*.

If you sell **B2C / high-velocity / dealership-style**: *The 10X Rule*, *Little Red Book of Selling*, *Pitch Anything*, *Cold Calling for Chickens*.

## Puerto Rico relevance

The modern strategy bucket maps cleanly onto PR's sales reality:
- **Hormozi's offer math** is what compresses the long-decision cycle PR buyers default to.
- **Voss's tactical empathy** is native — PR culture already rewards reading the room and slow-building trust.
- **Blount's pipeline discipline** is the cure for the relationship-heavy seller who confuses being liked with closing deals.
- **Cardone's 10X** translates directly to bilingual cold-outreach volume in a small-island TAM where 5,000 quality contacts is the whole market.
