# ⚙️ STRATEGIES

Cross-cutting frameworks distilled from the books and speaker libraries. Where the `BOOKS/` directory is organized by author, this directory is organized by problem.

Use `STRATEGIES/` when you need a working framework for a specific situation. Use `BOOKS/` when you want the full context behind it.

## Files

- **`psychology-of-persuasion.md`** — Cialdini's six principles, expanded with Carnegie, Bettger, and Voss applications.
- **`objection-handling.md`** — The four objection categories and the Ziglar / Hopkins / Voss responses to each.
- **`closing-frameworks.md`** — Assumptive, alternative, summary, sharp-angle, and Voss's calibrated close.
- **`prospecting-systems.md`** — Blount's cadence, Ross's predictable-revenue model, Hormozi's lead-gen four-quadrant.
- **`value-based-selling.md`** — Burg's Go-Giver laws, Willingham's integrity model, Pink's serving-not-selling reframe.
- **`offer-construction.md`** — Hormozi's value equation, Klaff's frame control, Holmes's dream-100.
- **`negotiation-tactical-empathy.md`** — Voss's full toolkit: mirroring, labeling, calibrated questions, the late-night-FM voice, accusation audit, Black Swan theory.

---

## How These Were Built

Each file is a synthesis, not a summary. The frameworks are stripped to the minimum viable kit — the question is always *what does a Puerto Rico operator do on Monday morning with this?*

Where two sources disagree, the disagreement is shown. Where a framework was popular but did not survive the field, it is flagged.
